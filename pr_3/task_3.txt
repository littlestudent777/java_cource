Какое значение будет в переменной c после выполнения данного кода?

int a = 1;
{
    int b = 2;
}
int c = a + b;

Ответ: При выполнении данного кода возникнет ошибка компиляции, и переменная c не примет какое-то значение, 
так как локальная переменная b не может быть доступна вне поля своей видимости. 
А полем её видимости является пространство внутри фигурных скобок.
