Итоговый вывод программы:
static logic
logic (1) id= 0
logic (2) id= 1
ctor id= 100

Пояснение:
  Класс A содержит следующие компоненты:
      Статический блок инициализации:
         static {
             System.out.println("static logic");
         }
      Он выполняется при загрузке класса и выполняется только один раз - при первом вызове конструктора или при попытке доступа к любому статическому члену класса.
      
      Поля класса:
         private int id = 1;
      
      Инстансные блоки инициализации:
         {
             System.out.println("logic (1) id= " + this.id);
         }
      
         {
             System.out.println("logic (2) id= " + id);
         }
      Эти блоки выполняются в порядке их объявления при создании нового экземпляра класса. Они выполняются до вызова конструктора.
      
      Конструктор:
         public A(int id) {
             this.id = id;
             System.out.println("ctor id= " + id);
         }

Порядок выполнения при создании объекта new A(100):

1. Статический блок:
   - Выводит: "static logic"

2. Инициализация полей значениями по умолчанию:
   - id получает значение 0

3. Первый блок инициализации:
   - Выполняется после инициализации полей по умолчанию
   - Выводит: "logic (1) id= 0"
   - id все еще равен 0, так как явная инициализация еще не произошла

4. Инициализация поля явным значением:
   - id = 1

5. Второй блок инициализации:
   - Выполняется после явной инициализации полей (id = 1)
   - Выводит: "logic (2) id= 1"

6. Конструктор:
   - Устанавливает id = 100
   - Выводит: "ctor id= 100"
