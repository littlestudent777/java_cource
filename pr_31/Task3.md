```markdown
В некоторых версиях Java (начиная с версии 15) появилась концепция текстовых блоков,
которые позволяют писать многострочные строки более удобнои.
Чтобы создать многострочный текст, используются тройные двойные кавычки:
```
```java
String multiLineString = """
    Это строка
    записанная в
    несколько строк.
    """;
```
