```java
public interface Interface {
  ...
}
```
```markdown
Интерфейс, объявленный с модификатором доступа public, имеет общедоступную область видимости.
Он может быть доступен и использован из любого другого пакета в данном Java-проекте.
То есть другим классам и интерфейсам разрешено реализовывать или наследовать этот интерфейс независимо от того,
в каком пакете они находятся.
```
