```java
private interface Interface {
  ...
}
```
```markdown
Интерфейс, объявленный с модификатором доступа private, имеет закрытую область видимости.
private интерфейс может быть доступен только внутри класса, в котором он объявлен.
Этот интерфейс недоступен для других классов, даже если они находятся в том же пакете.
```
