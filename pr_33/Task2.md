```markdown
Метод flush() используется, чтобы принудительно записать в целевой поток данные, 
которые могут кэшироваться в текущем потоке. Актуально при использовании буферизации 
и/или нескольких объектах потоков, организованных в цепочку.
```
```java
BufferedWriter writer = new BufferedWriter(new FileWriter("output.txt"));
writer.write("С помощью flush выполняется запись данных...");
writer.flush();  // Запись данных произведена!
writer.close();  // Ресурсы освободились.
```
