```java
try {
...
} catch (final Exception e) {
...
}
```
```markdown
Использование ключевого слова final в объявлении параметра исключения определяет переменную e
как неизменяемую внутри блока catch. Это значит, что после инициализации переменной e с
перехваченным исключением, больше нельзя изменить её значение в пределах этого блока.
```
