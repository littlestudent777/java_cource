```markdown
Можно ли использовать конструкцию try только с finally?
Да, в Java можно использовать конструкцию try только с блоком finally без блока catch. 
Блок finally выполнится независимо от того, произошло ли исключение в блоке try или нет.
Это полезно, например, для закрытия ресурсов, таких как файлы или соединения с базами данных,
которые необходимо освободить, даже если в процессе выполнения кода произошла ошибка.

Пример использования:
```
```java
public class Example {
    public static void main(String[] args) {
        try {
            System.out.println("Код, который может вызвать исключение");
            // Код
        } finally {
            System.out.println("Этот код выполнится всегда");
        }
    }
}
```
